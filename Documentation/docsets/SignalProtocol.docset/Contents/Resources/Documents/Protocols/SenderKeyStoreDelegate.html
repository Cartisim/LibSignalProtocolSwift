SenderKeyStoreDelegate

public protocol SenderKeyStoreDelegate

Implement the SenderKeyStoreDelegate protocol to handle the Sender Key storage of the Signal Protocol API. The keys should be stored in a secure database and be treated as unspecified data blobs.
